# Rank Products test

Performs rank products test comparing two groups Simplified
implementation for two-group comparison

## Usage

``` r
test_rankprod(control, treatment, n_perm = 1000)
```

## Arguments

- control:

  Control group values

- treatment:

  Treatment group values

- n_perm:

  Number of permutations for p-value estimation (default 1000)

## Value

p-value from the test
