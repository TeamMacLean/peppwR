# Plot QQ plots for goodness-of-fit

Plot QQ plots for goodness-of-fit

## Usage

``` r
plot_qq(fits, peptide_id = NULL, n_plots = 6)
```

## Arguments

- fits:

  A peppwr_fits object

- peptide_id:

  Specific peptide to plot (NULL for multiple)

- n_plots:

  Number of peptides to plot when peptide_id is NULL

## Value

A ggplot object
